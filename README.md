# BatraStore

An e-commerce website.

## Website Link

https://kartikbatra1808.pythonanywhere.com/

## Technologies

1. Python
2. Django
3. HTML
4. CSS
5. Bootstrap

## Installation

```bash
> git clone https://github.com/kartik-batra/BatraStore.git
> virtualenv .
> source bin/activate
> cd BatraStore
> pip install -r requirements.txt
> python manage.py runserver
```
